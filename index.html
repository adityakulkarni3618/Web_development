HTML Tutorial 23 & 24

HTML Entities

HTML entities are used to show special symbols.

Examples:

< = Less than

> = Greater than

& = Ampersand

" = Double Quote

© = Copyright

₹ = Rupee Symbol

Example sentence:

Price = 500 ₹


HTML Semantic Tags

Semantic tags give meaning to the webpage structure.

Examples of Semantic Tags:

<header> , <nav> , <section> , <article> , <footer>

Example:

This is Header

This is Navigation Area

This is Main Section

This is an Article

This is Footer
